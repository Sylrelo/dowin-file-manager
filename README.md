## Try it !

For now, if you want to try it you will need to build it from sources.

### Dependencies

* NodeJS 19.0 min, 20.10 recommanded.

### Executing program

#### Backend
```sh
cd backend
npm install
npm run watch
```
### Front
```sh
cd front
npm install
npm run dev
```
